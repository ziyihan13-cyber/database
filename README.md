# 综合仪表盘

京东竞品沙盘 & 电商新品跟踪 - 双页面数据可视化仪表盘

## 功能特点

- 📊 **京东竞品沙盘**：双月真实销量聚合分析
- 🆕 **电商新品跟踪**：区域销量价格分析
- 📁 **单文件上传**：一个Excel文件包含两个sheet，同时刷新两个页面
- 🔗 **商品链接跳转**：点击产品卡片的"查看"按钮可直接跳转商品页面

## 使用说明

1. 下载模板文件（包含两个sheet）
2. 填写数据后上传Excel文件
3. 两个页面会自动同步刷新

## 在线访问

部署后访问链接：`https://你的用户名.github.io/dashboard/`

## 本地开发

```bash
python -m http.server 8080
```

然后访问 `http://localhost:8080`
